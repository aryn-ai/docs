---
title: 'Async Partition Result'
openapi: 'get /v1/async/result/{task_id}'
---
This is the Aryn DocParse API for polling the status of an async task and getting its result when its done. For example, you can use [the async partition submit API](./async-partition-submit) to submit a document to be partitioned (and optionally chunked) asynchronously.
