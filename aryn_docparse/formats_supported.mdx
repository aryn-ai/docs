## Input Formats

The Aryn Partitioning Service supports the following input formats:

- `.pdf`
- `.docx`
- `.doc`
- `.pptx`
- `.ppt`
- `.csv`
- `.jpg` (`.jpeg`)
- `.png`
- `.bmp`
- `.tiff`
- `.html`
- `.odt`
- `.rtf`
- `.txt`
- `.xls`
- `.xlsx`
- `.xml`
- `.svg`
- `.webp`
- `.wmf`
- `.emf`
- `.mml`
- `.ods`
- `.xhtml`
- `.odp`
- `.odg`
- `.odf`
- `.ots`
- `.xltx`
- `.fods`
- `.xlt`
- `.slk`

## Output Formats

The Aryn Partitioning Service supports the following output formats:

- `.json`
- `.md`

For information, refer to the [output structure](./output) page.
