Aryn DocParse es un sistema de IA compuesto para analizar, fragmentar, enriquecer y almacenar documentos no estructurados a escala. Utiliza un conjunto de modelos de IA diseñados específicamente para la segmentación de documentos, reconocimiento óptico de caracteres (OCR), y extracción de tablas, imágenes, metadatos y más.

**Características Principales**

* Devuelve la salida estructurada de cada documento en JSON o Markdown, y proporciona cuadros delimitadores etiquetados para títulos, tablas, filas y columnas de tablas, imágenes y texto regular.

* Modelos de IA de alta calidad para extracción compleja de tablas, reconocimiento óptico de caracteres (OCR), resumen de imágenes y más.

* Procesa más de 30 tipos de formatos de documentos, incluyendo PDFs, Microsoft Word, Microsoft PowerPoint, texto y más.

* Almacena e indexa documentos procesados, extrae metadatos usando GenAI, busca en sus documentos a escala con búsqueda vectorial (semántica) o por palabras clave.

* Integración opcional con canalizaciones ETL de documentos en Python utilizando la [Sycamore document ETL library](https://github.com/aryn-ai/sycamore) de código abierto. Personalice su canalización con transformaciones de datos adicionales, extracción de entidades basada en LLM, enriquecimiento de datos, limpieza de datos y carga de bases de datos vectoriales y motores de búsqueda.

Puede usar DocParse para preparar datos complejos y no estructurados para aplicaciones de generación aumentada por recuperación (RAG), flujos de trabajo de procesamiento de documentos, extracción de contenido de documentos (como tablas) y sistemas de búsqueda semántica.

[Sign-up here for free](https://console.aryn.ai/signup/) para usar DocParse. Puede usar la interfaz de usuario de DocParse para visualizar su análisis y extracción, u obtener una clave API y usar el SDK de Aryn.

Puede aprender más de nuestro [introduction video](https://www.aryn.ai/?name=ArynPartitioningService_Intro)) o comenzar con una [Quickstart](docparse/quickstart).

Si está interesado en la Plataforma Aryn -- un almacén de datos no estructurados agéntico -- visite la [Aryn Platform documentation](./introduction). Aryn utiliza DocParse internamente para analizar y procesar documentos al ingerirlos.
