Aryn es un sistema agéntico de Procesamiento de Documentos y Análisis Profundo, que le permite consultar sus colecciones de documentos a escala con lenguaje natural. Puede mejorar la precisión (hasta 3 veces) y la eficiencia (hasta 10 veces) de los flujos de trabajo manuales como procesamiento, extracción y obtención de información de documentos complejos como informes financieros, memorandos legales, contratos, manuales técnicos y RFPs con diagramas técnicos. Aryn sirve como un Almacén de Datos No Estructurados Agéntico, permitiendo búsquedas y análisis complejos con lenguaje natural sobre documentos complejos a escala.

Aryn hace esto usando:

* Un modelo de razonamiento para "Análisis Profundo" — tareas complejas de múltiples pasos para extraer patrones e información de grandes conjuntos de documentos (hasta millones de documentos).
* Modelos de última generación para el análisis y ETL de documentos, incluyendo OCR, extracción de metadatos y tablas, y resumen en más de 30 formatos de documentos (por ejemplo, PDF, Word, PPT). Esto también está disponible de forma independiente con[DocParse](docparse/introduction).
* Potente interfaz de usuario de Espacios de Trabajo para navegar y consultar conjuntos de documentos para científicos de datos y analistas. Incluye Marcadores consultables para resultados intermedios, edición e inspección de planes de consulta, y organización de resultados.
* ETL de documentos personalizable con[integración de Sycamore](https://github.com/aryn-ai/sycamore)y APIs para construir aplicaciones y flujos de trabajo personalizados para usuarios finales.

También puede usar directamente DocParse ([ver video de introducción](https://www.aryn.ai/?name=ArynPartitioningService_Intro)), un componente básico de Aryn, para procesar sus documentos en JSON estructurado o Markdown. Puede usarlo para preparar datos complejos y no estructurados para aplicaciones de generación aumentada por recuperación (RAG), flujos de trabajo de procesamiento de documentos, extracción de contenido de documentos (como tablas) y sistemas de búsqueda semántica. Además, se integra con[Sycamore](https://github.com/aryn-ai/sycamore)para pipelines de ETL de documentos personalizados en Python.
